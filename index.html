Used Benchmarks & Datasets

Explore and download the standard datasets referenced in the base paper for evaluation.

FHM Dataset
A multimodal dataset containing synthetic memes with added confounding factors (no representative hateful metaphors in either single-modal visual features or textual features) designed to test advanced multimodal reasoning on targeted vulnerabilities.
HarMeme Dataset
Consists of COVID-19 related memes collected from Twitter, labeled with three classes: very hateful, partially hateful, and non-hateful. Used to study health crisis misinformation and public abuse patterns.
MAMI Dataset
Dedicated to identifying misogynistic content, focusing on several categories of offenses against women, including shaming, stereotype, objectification, and violence, providing labels for multiple visual-text sub-tasks.
HatReD Dataset
Hateful Memes with Reasons Dataset (HatReD) provides metaphor interpretation annotations for hateful memes. It includes human-annotated underlying reasons to explain implicit, context-heavy humor.

Base Paper Architecture Pipeline

The system models metaphor formation using three elements: metaphorical tenor, socio-cultural knowledge, and representation pattern.

Module 1
SCGen Search
Constructs prompts from 5 dimensions (race, nationality, gender, religion, disability) to extract visual objects, names, and retrieve socio-cultural knowledge.
Module 2
SCRA-MTI
Assesses socio-cultural relevance scoring (0, 1, or 2) between visual objects and extracted text, sorting them to identify the metaphorical tenor.
Module 3
Case Retrieval (RCR)
Searches Faiss vector database of human-annotated memes to retrieve similar representative case examples with explanations.
Module 4
CoT MLLM
Fuses tenor, knowledge, text, and retrieved cases in a Chain-of-Thought (CoT) prompt, yielding final safety classification and detailed reason.
